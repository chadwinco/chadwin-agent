# Python Setup (Fetch Japanese Company Data)

Use a local virtual environment in the repository root (`.venv/`).

## Create and activate
```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Install dependencies
```bash
python3 -m pip install -r requirements.txt
```

`yfinance` is required for this skill and is included in `requirements.txt`.
