# Source Log Format

Update `docs/source-log.md` when external sources are used.

## Table Format
`| Date | Company | Source | URL or File | Notes |`

## Rules
- Do not log local files under `companies/<TICKER>/data` as external sources.
- Add one row per external source used in research.
- In `Notes`, include the reason the source was used.

## Example
`| 2026-02-07 | CB | Earnings Call Transcript | https://example.com/transcript | Used to validate management commentary on pricing and reserves |`
