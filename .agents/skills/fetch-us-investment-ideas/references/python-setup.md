# Python Setup (Fetch US Investment Ideas)

Use a local virtual environment in the repository root (`.venv/`).

## Create and activate
```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Install dependencies
```bash
python3 -m pip install -r requirements.txt
```

`beautifulsoup4` and `requests` are required and already included in `requirements.txt`.
